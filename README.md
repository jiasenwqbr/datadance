## Intro 平台简介 

Efficient and fast completion of data governance, data exchange, data analysis, data visualization, privacy computing, etc
高效快速的完成数据治理、数据交换、数据分析、数据可视化、隐私计算等



## 前端开发注意事项

Node：建议v16或以上

安装依赖请支行：pnpm i

正常启动请运行: pnpm run dev

Mock测试模式请运行: npm run start

发布打包请运行: npm run build

## 后端开发
```
./startup.sh -m standalone
```
